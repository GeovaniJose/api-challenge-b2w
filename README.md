**api-challenge-b2w**
----
  Returns json data about of Star Wars planets.

* **Base URL**

  http://localhost:3001/v1

* **URL endpoints**

  `POST` /planets

  `GET` /planets

  `GET` /planets/name/:name

  `GET` /planets/:id

  `DELETE` /planets/:id

* **Method:**

  `GET`
  
*  **URL Params**

  `id=[integer]`
  
  `name=[string]`
